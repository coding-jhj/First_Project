# 젠스파크 PPT 제출 요건 반영 가이드

> 현재 미완성 PPT 기준으로 빠진 슬라이드만 빠르게 채우는 방법

---

## 젠스파크 기본 사용법

1. [genspark.ai](https://genspark.ai) 접속
2. 상단 메뉴 → **Spark Slides** 클릭
3. 프롬프트 입력창에 아래 내용 붙여넣기
4. 생성 후 슬라이드 클릭 → 텍스트/이미지 직접 수정 가능
5. 오른쪽 상단 **Export → PowerPoint** 로 저장

> **전체 16장 프롬프트**는 `genspark_ppt_prompt.md` 파일에 있음  
> 아래는 현재 PDF에서 **빠진 슬라이드만** 개별로 넣는 방법

---

Column 1 — 잘된 점 ✓ (green checkmarks)

· 거리 추정 정밀도 한계

· 거리 캘리브레이션·Depth 모델 검토
· 야간·혼잡 환경 검증

Bottom closing line (emphasized dark box):
"VoiceGuide는 보행 인프라를 대체하는 것이 아니라,
지금 앞의 위험을 한 번 더 알려주는 보조 안전 레이어입니다."

## 슬라이드 순서 권장 배치

현재 PDF 기준으로 아래 위치에 삽입:

| 삽입 위치 | 슬라이드 |
|---|---|
| 표지(1장) 바로 뒤 → **2장** | [프롬프트 A] 프로젝트 개요 |
| 사용자 조사(4장) 바로 뒤 → **5장** | [프롬프트 B] 고객여정지도 |
| 마지막 슬라이드 **바로 앞** | [프롬프트 C] 자체 평가 의견 |
| 마지막 슬라이드 | [프롬프트 D] 데모 영상 |

> 삽입 후 **목차 슬라이드의 번호**도 함께 업데이트할 것

# VoiceGuide 촬영 체크리스트

## 촬영 전 준비

- [ ] 앱이 설치된 Android 기기 준비
- [ ] 카메라 권한, 마이크 권한, 위치 권한 확인
- [ ] 서버 URL 설정 확인
- [ ] 서버 실행 확인
- [ ] 대시보드 접속 확인
- [ ] 화면 녹화 기능 확인
- [ ] 조용한 환경에서 TTS 소리 확인
- [ ] 의자, 문, 계단, 사람 등 테스트 객체 준비

## Android 앱 녹화

아래 장면을 각각 10~20초씩 2~3번 녹화합니다.

- [ ] 앱 실행 첫 화면
- [ ] 카메라 진입
- [ ] 의자 bbox 탐지
- [ ] 문 bbox 탐지
- [ ] 사람 bbox 탐지
- [ ] 계단 또는 위험 객체 bbox 탐지
- [ ] TTS 안내 장면
- [ ] 진동 패턴을 자막/아이콘으로 보여줄 장면
- [ ] 서버 업로드가 발생하는 장면
- [ ] 같은 세션이 대시보드에 반영되는 장면

## 서버/대시보드 녹화

- [ ] `POST /detect` 업로드가 발생하는 로그
- [ ] `POST /gps` 업로드가 발생하는 로그
- [ ] 대시보드에서 현재 객체 확인
- [ ] 대시보드에서 GPS 위치 확인
- [ ] 대시보드에서 탐지 이력 확인
- [ ] 필요 시 `/health` 또는 `/status/{session_id}` 확인 화면

## 촬영 품질

- [ ] bbox 라벨이 보일 만큼 화면이 큰가?
- [ ] 객체가 너무 어둡거나 멀지 않은가?
- [ ] TTS 음성이 녹음되는가?
- [ ] 자막이 앱 화면을 가리지 않는가?
- [ ] 영상에 불필요한 개인정보나 위치 정보가 보이지 않는가?

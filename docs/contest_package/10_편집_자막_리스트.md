# VoiceGuide 편집 자막 리스트

## 편집 기준

```text
최종 영상 길이: 5~10분
권장 길이: 7분 30초 안팎
출력: 1920x1080 / 30fps / MP4
```

## 타임라인

| 구간 | 시간 | 제목 |
|---|---:|---|
| 1 | 0:00~0:30 | 오프닝 |
| 2 | 0:30~1:20 | 문제 정의 |
| 3 | 1:20~2:00 | 서비스 개념 |
| 4 | 2:00~2:50 | 전체 구조 |
| 5 | 2:50~4:20 | 객체 탐지 |
| 6 | 4:20~5:10 | 음성·진동 안내 |
| 7 | 5:10~6:00 | 서버 업로드 |
| 8 | 6:00~6:50 | 서버와 대시보드 |
| 9 | 6:50~7:30 | 모델과 검증 |
| 10 | 7:30~8:20 | 시행착오 |
| 11 | 8:20~8:50 | 마무리 |

## 문제 정의

```text
길을 아는 것만으로는 부족합니다.
```

```text
중요한 것은 지금 앞의 위험입니다.
```

## 서비스 개념

```text
VoiceGuide는 길 안내를 대체하지 않습니다.
```

```text
지금 앞의 위험을 한 번 더 알려주는 보조 안전 레이어입니다.
```


# 실내 고정 물체 탐지 테스트 가이드

> 2026-05-07 미팅 기반 — 파이프라인이 물체를 제대로 판단하는지 검증하는 1차 테스트

---

## 목적

완전 정지 상태에서 하나의 고정 물체를 대상으로, YOLO + FastDepth + 진동/음성 파이프라인이 올바르게 작동하는지 확인한다.  
집 밖이 아닌 **실내**에서 진행해야 외부 변수(사람 이동, 빛, 노이즈)를 제거할 수 있다.

---

## 준비물

| 항목 | 내용 |
|---|---|
| 테스트 공간 | 실내 (방 또는 거실) |
| 물체 | 의자, 박스 등 — YOLO 학습 클래스에 포함된 것 |
| 고정 방법 | 물체를 바닥에 놓고 이동하지 않도록 고정 |
| 기기 상태 | 앱 실행 후 **사용자는 정지** |

---

## 테스트 절차

### Step 1 — 환경 세팅

1. 실내 조명 켜기 (충분한 밝기 확보)
2. 테스트할 물체 1개를 정면 **3m 이내** 바닥에 고정 배치
3. 주변에 다른 물체가 없도록 정리

### Step 2 — 거리별 탐지 확인

사용자는 물체 앞 각 거리에서 멈추고, 각 단계마다 아래를 확인한다.

| 위치 | 기대 동작 |
|---|---|
| 10m 앞 | 탐지 여부만 확인 (안내 없어도 됨) |
| 5m 앞 | 음성 안내 또는 진동 1회 |
| 3m 앞 | 음성 안내 + 진동 강조 |
| 1m 앞 | 긴급 경고 (진동 패턴 변화) |

> 이동할 때는 천천히 걷고, 각 지점마다 **완전 정지** 후 3~5초 대기

### Step 3 — 방향별 탐지 확인

같은 물체를 두고 아래 3가지 방향에서 각각 3m 거리에서 정지하여 탐지 여부 확인

- 정면
- 우측 비스듬히 (우전방)
- 좌측 비스듬히 (좌전방)

---

## 체크리스트

```
[ ] 물체가 BBox로 감지됨 (화면에 바운딩 박스 표시)
[ ] FastDepth 거리값이 실제 거리와 대략 일치
[ ] 3m 이내에서 진동 또는 음성 안내 발생
[ ] 같은 물체에 대해 반복 안내가 과하게 나오지 않음 (경고 피로 억제)
[ ] 정지 상태에서도 계속 탐지가 유지됨
[ ] 방향(정면/우전방/좌전방) 구분이 안내에 반영됨
```

## 주의사항

- 테스트 중 **사용자는 움직이지 않는다** — 움직임이 있으면 파이프라인 입력이 흔들려 오탐이 발생할 수 있음
- 동일 물체에 대해 음성 안내가 10초 이상 계속 반복된다면 **중복 안내 억제 로직** 점검 필요
- 진동 패턴이 거리별로 다르게 나오는지 별도로 메모

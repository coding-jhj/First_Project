# Presentation And Q&A

## 발표에서 강조할 점

- 사용자가 듣는 최종 안내 문장이 핵심 결과물입니다.
- 온디바이스와 서버 분석을 상황에 맞게 나눠 사용합니다.
- 탐지 정확도뿐 아니라 FPS, 응답 시간, 경고 피로도를 함께 관리합니다.
- 역할 분담과 검증 로그를 근거로 진행 상황을 설명합니다.

## 발표 준비 체크

- 앱 실행 흐름 설명 가능 여부
- 서버/GCP 요청 흐름 설명 가능 여부
- 실패 사례와 개선 방향 설명 가능 여부
- 팀원별 담당 파트 Q&A 준비

# Meeting Prep 2026-04-28

## 한 줄 요약

현재는 기능 추가보다 핵심 MVP와 팀 역할 설명을 안정화해야 합니다.

| 이름 | 역할 |
|---|---|
| 정환주 | 팀장, 서버, 프론트엔드 |
| 신유득 | Vision, ML |
| 김재현 | Android, UX |
| 임명광 | NLG, 서버 도움 |
| 문수찬 | Voice, Q&A 시트 작성 |

## 미팅에서 강조할 것

1. Android는 서버 없이도 온디바이스 ONNX로 장애물 탐지를 유지합니다.
2. 서버는 GCP Cloud Run의 `src.api.main:app` 하나로 통일합니다.
3. 오탐은 Vision/ML 실험표로 관리합니다.
4. FPS는 Android 담당자가 `VG_PERF` 로그와 병렬화 계획으로 설명합니다.
5. 실험 기능은 실험 기능으로만 말하고, 완료처럼 과장하지 않습니다.

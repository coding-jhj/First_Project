# VoiceGuide Docs

VoiceGuide 프로젝트 문서의 현재 사용 구조입니다.

## 폴더 구조

- `00_run/`: 로컬 실행, 명령어, 초기 설정
- `01_overview/`: 프로젝트 개요, 요구사항, 구조 설명
- `02_server/`: GCP, 배포, 서버 구조와 운영
- `03_android/`: Android 앱 성능과 온디바이스 처리
- `04_team/`: 팀 역할, 진행 기준, 담당 작업
- `05_presentation/`: 발표 자료, 스크립트, Q&A
- `06_debug/`: 탐지/성능 문제 분석과 해결 기록

## 참고

과거 회의록, 일일 정리, 긴 학습 문서, 실험 산출물은 `legacy/` 아래에 보관되어 있습니다.

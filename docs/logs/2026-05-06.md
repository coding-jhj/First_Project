## 핵심 이벤트

오늘은 **강사 미팅을 통한 아키텍처 전환 결정**과 그에 따른 전면 리팩터링이 이루어진 날.

1. **강사 미팅** — 서버 추론을 완전히 포기하고 온디바이스 전담으로 방향 전환 확정
2. **서버 추론 코드 완전 제거** (feature/jaehyun) — 이미지 수신·YOLO·Depth 경로 삭제
3. **걸음감지 자동 트리거 전환** — 가속도 자동 발동 제거, 질문 시 3초 요약으로 교체
4. **tracker AttributeError 버그 수정** — 매 `/detect` 요청이 500 반환하던 근본 원인 제거
5. **대시보드 세션 목록 수정** — GPS 없는 환경에서도 세션이 표시되도록 보완

## 강사 미팅 방향 전환 (핵심 결정사항)

> 오늘은 코드 수정이 아닌 **설계 방향 전환** 미팅. 아래 내용이 앞으로의 개발 기준이 됨.

**핵심 결론: 추론은 무조건 디바이스(폰)에서. 서버는 JSON 받아서 뿌리는 것만.**

| 항목 | 이전 | 변경 방향 |
|---|---|---|
| 추론 위치 | 서버 or 온디바이스 혼용 | **온디바이스 전담** |
| 서버 역할 | 추론 + 배포 | **JSON 수신 + 배포만** |
| 모델 | yolo11s | **yolo11n** (S는 폰에서 너무 무거움) |
| 클래스 수 | COCO 80개 | **필요한 몇 개만** 선택적 사용 |
| 거리 안내 | 이동 중 연속 안내 | **질문 시에만** (최근 3초 요약) |
| OBB | 검토 중 | **속도 문제로 보류** |

**이유:** 서버 추론 경로는 통신 지연 포함 FPS 6~7 → 실시간 기준(10fps) 미달. 구조적으로 해결 불가.

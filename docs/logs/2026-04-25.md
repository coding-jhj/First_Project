## 반영 실패 / 롤백 상세

**온디바이스 단독 방식 롤백**
- 시도: ngrok·서버 없이 폰에서 직접 ONNX 추론
- 실패 이유: 서버 없으면 보팅(Voting), 음성 파이프라인 전체, GPS 대시보드, Depth V2 등 다수 기능 확인 불가
- 조치: URL 입력 + OkHttp 서버 방식으로 복구 (커밋: `revert: 서버 방식으로 복구`)

## detect_objects 인터페이스 변경

```python
# 변경 전
def detect_objects(image_bytes: bytes) -> tuple[list[dict], dict]:
    nparr = np.frombuffer(image_bytes, np.uint8)
    img   = cv2.imdecode(nparr, cv2.IMREAD_COLOR)  # 매번 디코딩

# 변경 후
def detect_objects(image_input) -> tuple[list[dict], dict]:
    if isinstance(image_input, bytes):
        img = cv2.imdecode(np.frombuffer(image_input, np.uint8), cv2.IMREAD_COLOR)
    else:
        img = image_input  # 이미 디코딩된 numpy 배열 재사용
```

`detect_and_depth`에서 1회 디코딩한 `image_np`를 YOLO·Depth 양쪽에 재사용.
